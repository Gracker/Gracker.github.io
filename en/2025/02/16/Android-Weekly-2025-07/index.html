Android Weekly is a weekly newsletter focused on the Android technical ecosystem, updated every Monday. This newsletter delves into high-quality technical content in areas such as Android system architecture, performance optimization, cross-platform development, and AI, providing continuous knowledge updates and technical insights for developers.

Technical Articles

  1. Linux | Practical Causes of Process Zombie States: This article outlines real-world causes for zombie processes. These include child processes where the main thread exits but others are stuck in the D state, or parent processes using signalfd that get blocked by other epoll events, failing to reap children.

  2. Designing Memory Reclamation Strategies in Mobile OS from a Cybernetics Perspective: Exploring memory reclamation through closed-loop feedback control (like PID controllers). It analyzes using sampling and algorithms to create responsive memory management while emphasizing simplicity and scalability.

  3. Kotlin Weekly #446.

  4. Core Competitiveness for Techies: Thinking Beyond Code: Discussing principles developers should follow beyond mere coding, such as transcending requirements, processes, and silos to improve team efficiency and product quality.

  5. Detailed ANR Comparison: Android 13 vs. Android 10: A deep dive into source code differences between Android 10 and 13 regarding touch event ANRs, covering detection, judgment logic, and the treatment flow from Native to Java layers.

  6. Solutions for Fragment Transition Stuttering in Taobao Order Lists: Transitioning from “known patterns” to high uncertainty, Taobao shares its hybrid architecture (Native containers + Weex 2.0) to solve experience issues by leveraging native performance and gesture recognition.

  7. Experience and Insights from Taobao Page First Frame Optimization: Sharing performance analysis, metrics, and specific strategies for optimizing first-frame delivery on mobile pages to enhance UX and reduce costs.

  8. Exploration of Taobao App Transaction Chain Hybrid Scenes: Further practice from the “Taobao Transaction Architecture” series, mixing Native and dynamic languages for better delivery speed and quality.

  9. Chronicle of Weex Cart Long-List Horizontal Swipe Optimization: Reviewing the iterative journey of optimizing horizontal swiping in the Weex-based Taobao cart.

  10. Correctly Managing Binder Object Lifecycle in One Sentence: Avoiding memory leaks and thread safety issues in Binder communication with detailed code examples.

  11. Simple Overview of Android View Drawing Flow: Explaining the hierarchy and responsibilities in Activity View drawing, including logic for measure, layout, and the six steps of draw.

  12. Deep Understanding of Jetpack Lifecycle (Principle Part): Analyzing LifecycleOwner, Observer, and the Registry mechanism, explaining how Activity and Fragment implement lifecycle awareness.

  13. The Second Beta of Android 16: Introducing pro camera features, graphics supports, and performance framework extensions in the latest beta.

  14. Dart 3.7 Released: What’s New?: While standard Macro progress is paused to focus on performance, Dart 3.7 reinvests in data implementation and augmentations to support better serialization.

  15. Flutter’s Widget Key Proposals: Talking Key Roles: Discussing how Key objects maintain state and facilitate widget reuse in the Element tree.

  16. Debugging and Fixing an App Crash Caused by Accessibility Testing: Investigating crash issues caused by accessibility automation, leading to improved logging and root cause analysis.

  17. Flutter 3.29 Released: Bringing updates to Cupertino, Material, and Web, including engine optimizations like Backdrop filter improvements.

  18. Interaction Mechanism between WM and SF — Transaction: Exploring how Transaction acts as a bridge between Windows Manager and SurfaceFlinger, reducing IPC and ensuring synchronous transformations.

  19. TrustedTime API: A Reliable Approach to Time Keeping: A new tool for tamper-proof timestamps backed by Google’s infrastructure.

  20. Technical Briefing 2025 Issue 2: Roundup of C++, system design, MESI protocol, and eBPF resources.

  21. Android 15 App Cold Start (1) Activity Lifecycle: Analyzing ATMS requests and Task management during cold starts of heavy games like Honor of Kings.

  22. Android 15 App Cold Start (2) Window Hierarchy Construction.

  23. Android 15 App Cold Start (3) Adding Startup Windows: Detailed analysis of data packaging and multithreaded View creation for startup windows.
